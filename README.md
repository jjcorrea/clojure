[Clojure][1]
===============

- Emphasis in immutability and referential transparency (no side-effects)
- Explicit concurrency support
- Designed for the JVM

Clojure is a LISP
=================

- Code as data
- Dynamically typed, and compiled
- Interactive - REPL
- Load / Change code in running program
